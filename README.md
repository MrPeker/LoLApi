# Neptune Framework


### A PHP Framewok

### Requirements

  - PHP >= 5.6
  - Composer

### Installation

Clone or download this repository. 
And go to your project directory

```sh
$ composer install
$ composer dump-autoload -o
$ php neptune --secret
```
